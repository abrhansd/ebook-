# 🚀 Landing Page - Domine sua Rotina com IA

Este repositório contém o código-fonte da Landing Page oficial de vendas para o e-book **"Domine-sua-rotina-programe-menos-e-produza-o-dobro-com-IA.pdf.pdf"**. 

O site foi desenhado com uma estética moderna, limpa e responsiva, utilizando gatilhos mentais e dados de alto impacto baseados no conteúdo do próprio guia para maximizar as conversões.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído focando em performance de carregamento e facilidade de manutenção:

* **HTML5**: Estruturação semântica de toda a página.
* **Tailwind CSS**: Framework utilitário usado via CDN para um design rápido, moderno e totalmente responsivo (mobile-first).
* **Font Awesome**: Biblioteca de ícones vetoriais usada para ilustrar os benefícios e seções de forma intuitiva.

---

## 💎 Estrutura da Página

1. **Hero Section:** Chamada principal focada na dor do cliente (perda de tempo com tarefas manuais) e botão de ação direta.
2. **Prova Social & Estatísticas:** Destaques visuais sobre os benefícios da automação (economia de tempo e aumento de produtividade).
3. **Público-Alvo:** Segmentação clara para profissionais autônomos, estudantes de tecnologia, programadores iniciantes e entusiastas de produtividade.
4. **Resumo dos Capítulos:** Demonstração do valor do conteúdo (Mentalidade de Automação, Ferramentas Essenciais, Engenharia de Prompt e Projetos Práticos).
5. **Seção de Oferta (CTA):** Bloco de fechamento integrado diretamente com o link de pagamento da Kiwify.
